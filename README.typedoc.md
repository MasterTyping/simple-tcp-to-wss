> Node.js 기반 TCP to WSS 브릿지 서버의 API 문서입니다.

---

## 📖 소개

- TCP 서버에서 데이터를 받아 WebSocket(WSS)으로 브라우저에 전달하는 Node.js 서버입니다.
- TypeScript 기반으로 작성되어 있습니다.
- 주요 클래스: `SocketServer`
- 주요 타입: `SocketClient`, `Subscription`
